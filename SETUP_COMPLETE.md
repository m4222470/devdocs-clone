## 🚀 كيفية البدء

### الخطوة 1️⃣: تشغيل الخادم

```bash
cd /workspaces/devdocs-clone
./run_server.sh
```

**أو يدويًا:**
```bash
cd /workspaces/devdocs-clone/sentrixa-template.webflow.io
python3 -m http.server 8000
```

### الخطوة 2️⃣: فتح المتصفح

```
http://localhost:8000
```
