# Devdocs Clone - Documentation Website Template

A clean, modern documentation website template converted from Webflow to pure HTML/CSS/JS. No external dependencies, no tracking scripts, fully local and self-contained.

## Features

- ✅ Pure HTML/CSS/JS - No frameworks required
- ✅ Responsive Design - Works on all devices
- ✅ Dark/Light Mode - Toggle between themes
- ✅ Search Functionality - Full-text search
- ✅ Sidebar Navigation - Easy doc organization
- ✅ Clean Animations - Smooth transitions
- ✅ SEO Optimized - Meta tags included
- ✅ Accessible - ARIA labels and semantic HTML
- ✅ No External Dependencies - 100% local

## Pages

- Home - Landing page
- Documentation - Multiple docs pages
- Help Center - Support pages
- Changelog - Updates and releases
- Support - Contact and help

## Usage

Simply open `index.html` in any browser. No server required.
