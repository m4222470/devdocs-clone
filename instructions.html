١. التثبيت

٢. التخصيص

يمكنك تخصيص المشروع عبر تعديل الملفات التالية:

  • ملفات HTML - المحتوى

٣. النشر

للنشر على Netlify:

  1. ارفع المستودع على GitHub
  2. اربط المستودع بـ Netlify
  3. سيتم النشر تلقائياً
