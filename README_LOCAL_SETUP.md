## 🚀 كيفية تشغيل الموقع محليًا

### الطريقة 1️⃣: استخدام البرنامج النصي

```bash
cd /workspaces/devdocs-clone
./run_server.sh
```

ثم افتح المتصفح على:
```
http://localhost:8000
```

### الطريقة 2️⃣: تشغيل الخادم يدويًا

```bash
cd /workspaces/devdocs-clone/sentrixa-template.webflow.io
python3 -m http.server 8000
```

### الطريقة 3️⃣: استخدام Node.js (إن كان مثبتًا)

```bash
cd /workspaces/devdocs-clone/sentrixa-template.webflow.io
npx http-server -p 8000
```

## 🎨 الأسلوب والتصميم

الموقع يحتوي على:

- ✅ **تصميم قابل للاستجابة** - يعمل على جميع الأجهزة
- ✅ **تنسيق احترافي** - مع ألوان وخطوط محسنة
- ✅ **حركات سلسة** - في الملاحة والانتقالات
- ✅ **تحميل سريع** - جميع الموارد محلية
- ✅ **لا يعتمد على الإنترنت** - يعمل بدون اتصال بعد التحميل

## 🔗 الروابط والملاحة

جميع الروابط تم تحويلها للعمل محليًا:

- ✅ الروابط الداخلية (بين الصفحات) تعمل بشكل صحيح
- ✅ الصور والموارد تحميل من المجلدات المحلية
- ✅ ملفات CSS و JavaScript محلنة بالكامل
- ✅ لا توجد روابط خارجية متكسرة

## 🛠️ استكشاف الأخطاء

### المشكلة: الموقع لا يحمل الصور

**الحل:** تأكد من تشغيل الخادم من المجلد الصحيح:
```bash

python3 -m http.server 8000
```

### المشكلة: الروابط المكسورة

**الحل:** تأكد من أن كل الملفات موجودة بتشغيل:
```bash
./verify_website.sh
```

### المشكلة: CSS لا يحمل

**الحل:** تأكد من وجود مجلد `../cdn.prod.website-files.com/` بنفس مستوى مجلد الموقع.
