أمثلة أكواد

مجموعة شاملة من أمثلة الأكواد الجاهزة للاستخدام.

JavaScript

أمثلة أكواد JavaScript للتكامل مع API:

JavaScript 
// تهيئة API
const api = new DevDocsAPI({
  apiKey: 'your-api-key',
  baseUrl: 'https://api.example.com'
});

// الحصول على قائمة المستخدمين
async function getUsers() {
  try {
    const response = await api.get('/users');
    return response.data;
  } catch (error) {
    console.error('خطأ:', error);
    throw error;
  }
}

// إنشاء مستخدم جديد
async function createUser(userData) {
  const response = await api.post('/users', userData);
  return response.data;
}

// تحديث مستخدم
async function updateUser(id, userData) {
  const response = await api.put(`/users/${id}`, userData);
  return response.data;
}

// حذف مستخدم
async function deleteUser(id) {
  await api.delete(`/users/${id}`);
}

Python

أمثلة أكواد Python للتكامل مع API:

Python 
import devdocs

# تهيئة API
api = devdocs.API(api_key='your-api-key')

# الحصول على قائمة المستخدمين
def get_users():
    try:
        response = api.get('/users')
        return response['data']
    except devdocs.APIError as e:
        print(f'خطأ: {e}')
        raise

# إنشاء مستخدم جديد
def create_user(user_data):
    response = api.post('/users', user_data)
    return response['data']

# تحديث مستخدم
def update_user(user_id, user_data):
    response = api.put(f'/users/{user_id}', user_data)
    return response['data']

# حذف مستخدم
def delete_user(user_id):
    api.delete(f'/users/{user_id}')

cURL

أمثلة أوامر cURL للتعامل مع API:

bash 
# الحصول على قائمة المستخدمين
curl -X GET https://api.example.com/v1/users \
  -H "Authorization: Bearer your-api-key"

# إنشاء مستخدم جديد
curl -X POST https://api.example.com/v1/users \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "أحمد محمد",
    "email": "ahmed@example.com"
  }'

# تحديث مستخدم
curl -X PUT https://api.example.com/v1/users/123 \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "أحمد الجديد"
  }'

# حذف مستخدم
curl -X DELETE https://api.example.com/v1/users/123 \
  -H "Authorization: Bearer your-api-key"
→ السابق إنشاء حساب التالي ← الدعم الفني
